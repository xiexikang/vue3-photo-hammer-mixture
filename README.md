# vue3-photo-hammer-mixture
## 这是个案例，下载下来根据自行情况进行修改。
 案例CSDN链接：[案例链接][1]
功能：（挂饰拖动滑动，缩放，旋转，重置位置，合成图片，预览图片）

 - 环境：vue3.x+vite2 (根据自身环境进行修改)
 - ui框架：vant3 （根据自身自行更换）
 - 手势库：hammerjs （必需）
 - 截图库：html2canvas （必需）


## Project setup
```
yarn
```

### run
```
yarn run dev
```

### UI
```
vant3
```
